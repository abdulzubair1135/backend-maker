# Professional CMS & API Core System

A robust, enterprise-grade Content Management System (CMS) and API Backend powered by **Node.js, Express, and MySQL**. Complete with a glassmorphic dark-mode Admin Control Panel and advanced security configurations.

## 📂 Project Structure

- **`Frontend/`** - The dynamic customer-facing website (pre-patched to consume backend APIs).
- **`Backend/`** - Express.js REST API server with JWT Auth, Role-Based Access Control, and Security middleware.
- **`Database/`** - SQL initialization schemas and database configurations.
- **`Admin/`** - Glassmorphic Admin Control Panel for CMS management, leads checking, logs, media, and backups.
- **`Uploads/`** - Static media directory.
- **`Logs/`** - Standard error and system access logs.
- **`Backups/`** - Local database serialization dumps.
- **`Documentation/`** - In-depth Deployment Guides and API endpoints specs.
